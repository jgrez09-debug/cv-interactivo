# CV Interactivo — Jonathan Grez Parada

Currículum Vitae interactivo desarrollado como **Evaluación Final del Módulo 2**.

## 🔗 Demo en vivo
[Ver CV en GitHub Pages](https://jgrez09-debug.github.io/cv-interactivo/)

## 🎯 Objetivo
Desarrollar una aplicación web con HTML, CSS, Bootstrap, JavaScript y jQuery que sea completamente responsiva, contenga efectos interactivos y esté alojada en GitHub.

## 🛠️ Tecnologías utilizadas
- **HTML5** — Estructura semántica con etiquetas section, nav, header, footer, form
- **CSS3** — Variables CSS, animaciones, diseño responsivo sin frameworks adicionales
- **Bootstrap 5** — Grid system, navbar responsiva, utilidades de espaciado
- **JavaScript ES6+** — IntersectionObserver, manipulación del DOM, localStorage
- **jQuery 3.7** — Eventos onChange, onClick, animaciones slideDown/slideUp

## ✨ Funcionalidades interactivas (4 eventos implementados)
1. 🌙 **Modo claro / oscuro** — Toggle en la navbar, guarda preferencia en localStorage
2. 📜 **Animación al hacer scroll** — Secciones y tarjetas se revelan con efecto fade+slide
3. ✅ **Formulario con validación en tiempo real** — Valida nombre, email, asunto y mensaje con jQuery onChange/onInput
4. 🎨 **Cambio de color de acento** — Botón en Experiencia que rota entre 6 paletas premium

## 📁 Estructura del proyecto
```
cv-interactivo/
├── index.html    ← Estructura HTML semántica
├── style.css     ← Estilos, variables CSS y animaciones
├── script.js     ← Interactividad con JavaScript y jQuery

```
